Sistema Administrativo (SIAD) 1.0
Colegio de Sociólogos de Huánuco
Sistema web interno para la gestión integral de colegiados, control de pagos, deudas, egresos y generación de reportes administrativos.

Características Principales

✅ Gestión completa de colegiados
✅ Control de pagos e ingresos
✅ Administración de deudas y morosidad
✅ Registro de egresos institucionales
✅ Reportes financieros detallados
✅ Control de habilitación automática/manual
✅ Búsqueda pública de colegiados habilitados
✅ Sistema de roles y permisos
✅ Importación masiva desde Excel


Tecnologías Utilizadas

Backend: PHP 7.4+
Base de Datos: MySQL 5.7+
Frontend: HTML5, CSS3, JavaScript
Arquitectura: MVC + Capas
Control de Versiones: Git
